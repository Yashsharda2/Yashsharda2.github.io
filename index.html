Projects

FIR & FFT Signal Processing Engine [GitHub]

Implemented high-performance DSP blocks using fixed-point arithmetic for real-time signal analysis.

  • Designed a 16-tap low-pass FIR filter with symmetric coefficients for linear phase response.
  • Implemented a 16-point Radix-2 DIF FFT utilizing Q1.15 fixed-point arithmetic to minimize quantization error.
  • Optimized butterfly structures for resource efficiency on FPGA fabric, ensuring high-accuracy rounding across both modules.

Real-Time Image Processing Pipeline [GitHub]

Building a real-time image processing pipeline on Vaaman SBC using Verilog.

  • Implemented core operations including brightness adjustment, grayscale conversion, and binary thresholding.
  • Designed a pipelined Sobel edge detector on a 400-wide grayscale stream using dual line buffers and Manhattan gradient approximation.

SPI Communication Protocol Implementation [GitHub]

Implemented a high-speed, full-duplex SPI Master controller in Verilog.

  • Developed a synthesizable SPI state machine, achieving a stable 5MHz SCLK from a 50MHz PLL-generated system clock.
  • Designed a physical loopback interface on the Vaaman board, synchronizing RTL signals with SoC GPIOs.
  • Achieved reliable 8-bit data transfers between the FPGA fabric and the RK3399 SoC.

RP2040 Pico Music Player [GitHub]

  • Built a WAV audio player using DAC/PWM-based output on the RP2040.
  • Interfaced rotary encoders and an SPI display for a real-time user interface.
